1. Install Node.js https://nodejs.org/en/download/

2. To install packages execute 
    npm install

3a. To build project during development with hot reload execute
    npm run build

3b. To build project for production  execute
    npm run dist


This template is designed to migrate a classic website layout to be used with Webpack.

There is a config file for development with Hot Module Reload, as well as a distribution file with several plugins for optimisation purposes.


After running Webpack, the distribution folder has this structure:
- fonts/
- images/
- bundle.js
- index.html
- styles.css
